```

```

## 🎯 最も成功率が高い形式

### 1. エラーメッセージ提示
```
このエラーを直して:
NameError: name 'User' is not defined
```

### 2. 具体的ファイル・行番号
```
backend/app/models/user.py:45
```

### 3. Before/After形式
```
Before: user = get_user()
After: user = await get_user()
```
